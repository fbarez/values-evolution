# Serialization Decisions Log

Document every serialization variant tried, the rationale, and the result.
This log is the audit trail for ablation decisions.

## Format

Each entry:
- **Variant**: config name
- **Config hash**: first 12 chars of SHA256
- **Date**: when run
- **Val JS divergence**: result on validation set
- **Decision**: selected / rejected / pending
- **Rationale**: why
